# Intellilight
Adaptive smart bulb controller using ESP32, proximity sensor, ambient light sensor and colour sensor

This project was designed for the _Element14_
[**Summer of Sensors Design Challenge**](https://community.element14.com/challenges-projects/design-challenges/summer-of-sensors-design-challenge/w/documents/27682)
under the challenge title: **Exploration Station**

For regular updates during the competition please follow the blogs on my [wiki](https://github.com/Mayannaise/intellilight/wiki)
